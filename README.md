#  ESIMedia – Plataforma de Streaming de Contenidos Multimedia

##  Descripción
ESIMedia es una plataforma web de **streaming de contenidos multimedia** (audio y vídeo) desarrollada como práctica integrada de las asignaturas:

- **Gestión de Proyectos Software**
- **Seguridad de Sistemas Software**
- **Procesos de Ingeniería del Software**
- **Calidad de Sistemas Software**


---

##  Roles del sistema
- **Administrador**: gestiona usuarios, con funciones de creación, modificación, bloqueo/desbloqueo y eliminación.  
- **Creador de contenido**: sube y gestiona contenidos, controla su disponibilidad, consulta estadísticas y valoraciones.  
- **Usuario**: reproduce, valora contenidos y gestiona listas y favoritos.  
  - **Estándar**
  - **VIP**

##  Objetivos principales
- Asegurar un **código robusto y seguro**, con cobertura >60% y sin vulnerabilidades críticas.  
- Garantizar **disponibilidad y escalabilidad** en el servicio.  
- Cumplir con las normas y buenas prácticas de **seguridad y calidad de software**.  
- Proporcionar una **experiencia de usuario completa** en web y móvil.
