# Requisitos del Proyecto

Este directorio contiene los documentos relacionados con los **requisitos** del sistema.

## Subcarpetas

- **01_recogida_inicial/**  
  Documentos de la primera recogida de requisitos y entrevistas de elicitación.

- **02_Historias_de_usuario**  
  Historias de Usuario (HU), criterios de aceptación y trazabilidad con requisitos.

- **03_Diagramas/**  
  Diagramas UML, casos de uso, etc..

- **04_Reuniones_-_Clases_Recogida_y_Cuestiones/**  
  Aquí podrá encontrar todos los documentos que contienen información relevante en clase sobre requisitos.
